All Keys Lost Solutions

When every single remote, transponder fob, or mechanical key to a vehicle has vanished, you don't need a costly dealership tow. Our mobile workshop is configured to decode lock cylinders, generate code profiles, and program fresh data directly into the vehicle's immobiliser system completely from scratch at the roadside.

Our Mobile Recovery Sequence

An "All Keys Lost" situation requires a precise blend of physical lock decoding and software level EEPROM manipulation. Here is exactly how we generate replacement keys right where the vehicle sits:

1. Non-Destructive Mechanical Decoding

Using precision Lishi picking tools, we manipulate the internal wafer configuration of the door or ignition lock cylinder. Once turned, we interpret the internal depths of the wafers to decode the original mechanical key cut code without inflicting a single scratch on the lock assembly.

2. Computerised Precision Milling

The decoded key code is entered into our automated CNC milling machine. The laser cutter traces a brand-new, factory-standard key blank down to exact millimeter factory tolerances, counteracting any wear and tear found in the old locks.

3. Transponder & Transceiver Linking

A cut blade will only turn the ignition; it won't clear the engine immobiliser. We interface directly with the vehicle's diagnostic port (OBD) using advanced programming suites to clear out the old lost keys for security, extract the required pin-codes, and pair brand-new transponder chip data to the engine control module.

Technical Capabilities

  • Mechanical Lock Decoding: Full decoding configurations for HU66, HU101, HU162T, and NSN14 keyways.
  • Key De-authorisation: Complete deletion of missing/stolen keys from the vehicle memory database.
  • Transponder Matching: Support for ID46, ID48, Megamos Crypto, and DST-AES electronic configurations.
  • Commercial Vans Covered: Specialized field workflows tailored for Ford Transit Custom, VW Transporter, and Vauxhall Vivaro fleets.
